# Web Snake Game

An Snake Game written with pixi.js

## Run
```
yarn start-dev
```
or paly at <br/>
[http://karangoel.codes/games/snake/index.html](http://karangoel.codes/games/snake/index.html)

## How to play

Arrow keys to move <br/>
P to pause <br/>
Esc to Menu <br/>
